## How to run

type
```bash
mvn clean verify -Dtestng.dtd.http=true -Dmaven.test.failure.ignore=true
```
in terminal

## Main Features
* Selenium
* Java
* TestNG
* Page Object Model
* [Log4j 2](https://logging.apache.org/log4j/1.2/apidocs/org/apache/log4j/PatternLayout.html)
* Capture Screenshots
* Cucumber BDD Framework
* Cucumber HTML Report
* Allure Report
